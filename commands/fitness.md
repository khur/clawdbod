# Fitness Break

Use the clawdbod skill to give the user an exercise challenge right now.

If $ARGUMENTS contains "hiit 5", generate a 5-minute HIIT workout.
If $ARGUMENTS contains "hiit 10", generate a 10-minute HIIT workout.
If $ARGUMENTS contains "summary" or "stats", show the session fitness summary.
Otherwise, do a quick micro-break challenge (pick a random exercise and ask for reps).
